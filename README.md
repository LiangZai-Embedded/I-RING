# I-RING V1.0  
### 演示视频 https://www.bilibili.com/video/BV1Ca411h7yj?spm_id_from=333.999.0.0

## 硬件  

1.主控: STM32F030K6T6 ARM Cortex-M0@48MHz  

2.imu单元: ICM20602六轴传感器 HMC5883L磁力计  

3.充电芯片:TP4056  

4.充电接口:Micro USB  

5.显示器:SSD1306 I2C OLED  

## 软件  
1.GUI  u8g2单色图形库  

2.功能：电池电量，RTC,计步，秒表，闹铃，俯仰横滚角，指南针，游戏
