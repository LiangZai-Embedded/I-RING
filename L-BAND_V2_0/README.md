# L-BAND V2.0  
### 演示视频   



## 硬件  

1.主控: STM32G031F8P6 ARM Cortex-M0+ @64MHz  

2.IMU: LSM303DLHC  

3.气压计/温度计:BMP280  

4.电池:401120-80mAH锂聚合物电池  

5.显示器:0.96' SSD1306 OLED  


## 软件  
1.GUI  u8g2单色图形库  

2.功能：电池电量,日历,温度,计步,秒表,俯仰横滚角,指南针,游戏 

3.开发环境  STM32CubeIDE  

### 功耗：正常模式约8mA,待机模式约0.4mA，后续进一步降低
